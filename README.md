# ReboundLabs Website

A premium, high-performance web application for ReboundLabs, featuring a "Midnight Premium" dark aesthetic, animated backgrounds, and glassmorphism UI.

## Features
- **Modern Dark Theme**: Deep slate backgrounds with neon accents.
- **Interactive UI**: Glassmorphism cards, spotlight effects, and expandable service details.
- **Animations**: Entrance animations, hover effects, and parallax scrolling.
- **Responsive Design**: Fully optimized for all devices.
- **Contact Form**: Integrated with Formspree for handling submissions.

## Tech Stack
- React
- Vite
- CSS Modules / Vanilla CSS

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the local server.
